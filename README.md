# Simple Interest Calculator

## About
This project provides a simple Bash script to calculate 
simple interest for micro-finance applications.

## Purpose
To help low income individuals understand and calculate 
simple interest on loans or savings.

## Usage
The script takes the following inputs:
- **Principal** : The initial amount of money
- **Rate of Interest** : The interest rate applied
- **Time Period** : The duration of the calculation
